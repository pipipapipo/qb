# QB Engineering & Construction Static Website

This is an original static HTML/CSS/JS website for GitHub Pages.

## Pages
- index.html
- about.html
- services.html
- projects.html
- contact.html
- assets/css/style.css
- assets/js/main.js
- assets/images/

## What to change
Search for these placeholders:
- contact@qbengineers.com
- +63 912 345 6789
- YOUR-PAGE
- YOUR-HANDLE
- YOUR-FACEBOOK-PAGE

## Images
Replace the SVG placeholders inside assets/images with your real files.
You can keep the same filenames or update the `src` values in the HTML files.

## GitHub Pages
Upload all files to your repository root or `qb/` folder.
Make sure `index.html` is at the publishing folder root.
